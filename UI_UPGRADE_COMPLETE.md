## 🎨 Design System

### **Color Palette:**

**Primary Colors:**
- Purple: `#667eea`, `#6366f1`
- Indigo: `#764ba2`, `#8b5cf6`
- Pink: `#ec4899`
- Green: `#10b981`, `#059669`

**Neutral Colors:**
- Dark: `#0f172a`
- Gray: `#64748b`
- Light Gray: `#f1f5f9`
- White: `#ffffff`

**Accent Colors:**
- Blue: `#dbeafe`, `#1e40af`
- Yellow: `#fef3c7`, `#92400e`
- Purple Light: `#ede9fe`

### **Typography:**

**Headings:**
- H1: 2.5rem, font-weight: 800
- H2: 1.5rem, font-weight: 700
- H3: 1.125rem, font-weight: 600

**Body:**
- Regular: 1rem
- Small: 0.875rem
- Tiny: 0.75rem

### **Spacing:**
- Small: 0.5rem (8px)
- Medium: 1rem (16px)
- Large: 1.5rem (24px)
- XL: 2rem (32px)

### **Border Radius:**
- Small: 0.375rem
- Medium: 0.5rem
- Large: 0.75rem
- XL: 1rem

### **Shadows:**
- Card: `0 4px 6px rgba(0, 0, 0, 0.1)`
- Hover: `0 8px 12px rgba(0, 0, 0, 0.15)`
